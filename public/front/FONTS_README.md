# 字体说明

请下载以下字体并放置在此目录：

## Orbitron (科技感字体，用于标题)
- 下载地址：https://fonts.google.com/specimen/Orbitron
- 需要文件：
  - Orbitron-Regular.ttf
  - Orbitron-Bold.ttf

## JetBrains Mono (编程字体，用于正文)
- 下载地址：https://www.jetbrains.com/lp/mono/
- 需要文件：
  - JetBrainsMono-Regular.ttf

或者你可以使用已有的字体文件（UnidreamLED.ttf 或 Pacifico-Regular.ttf）

如果字体文件缺失，系统会回退到默认字体。
